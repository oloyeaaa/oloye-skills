## Install (60 seconds)

Inside [Claude Code](https://claude.com/claude-code), add the marketplace once:

```text
/plugin marketplace add oloyeaaa/oloye-skills
```

Then install whichever skill you want (from the table above), for example:

```text
/plugin install invoice-chaser@oloye-skills
```

Now use it by typing its name:

```text
/invoice-chaser
```

It'll ask you a couple of quick questions and get to work. That's the whole thing.

> New skills get added to this same marketplace over time. Pull the latest with:
>
> ```text
> /plugin marketplace update oloye-skills
> ```

## Requirements

- [Claude Code](https://claude.com/claude-code) installed and signed in.
- For **Invoice Chaser** you connect your Gmail (it drafts, it never sends). For **Post Pack** scheduling you
  connect Metricool (optional). The skill tells you when it needs something.
- That's the whole list.

Want to read exactly what a skill does before installing? Open its `SKILL.md` under
[`plugins/`](plugins/) — plain English, no surprises.
